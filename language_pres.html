LISP - programming language

Origin

LISP was specified in 1958 by John McCarthy at MIT, making it the second oldest programming language still used. It's name comes from the term "List Processor", used because linked lists are the main data structure in LISP.

Uses

LISP is closely tied with early exploration into AI or artificial intelligence systems, and was used to write the programming language of SHRDLU, one of the most successful "interactive fiction" AI of the time.

Pioneering ideas

As one of the earlist languages, LISP was a trailblazer. Many of the fundamentals of computer science today, such as tree-style data structures, conditionals (i.e. booleans), higher order functions (functions with functions as arguments), and recursion got their start in LISP.

Syntax and example

LISP is known by it's parentheses heavy syntax. All programs, code, and data are written as "s-expressions", unlike in other languages which make regular use of brackets and braces. This has given LISP nicknames like "Lost in stupid parentheses", although this regularity also gives LISP much of it's power. Below is a function in LISP that finds the factorial of a number. (the product of all positive integers less than or equal to a given number)

(defun factorial (n) (if (= n 0) 1 (* n (factorial (- n 1)))))
